# 📖 Tentang Project

**Temperature Conversion Using Dart** merupakan aplikasi berbasis **Dart Console** yang digunakan untuk melakukan konversi suhu antar berbagai satuan.

Project ini saya kerjakan ketika masih menempuh pendidikan di **SMK Negeri 1 Pacitan** jurusan **Rekayasa Perangkat Lunak (RPL)** sebagai latihan dasar pemrograman menggunakan bahasa **Dart**.

Repository ini saya simpan sebagai dokumentasi perjalanan belajar saya dalam memahami logika pemrograman, operasi matematika, dan penggunaan bahasa Dart.

---

# ✨ Fitur

Program menyediakan proses konversi suhu secara sederhana melalui aplikasi console.

Fitur utama:

* 🌡️ Input nilai suhu.
* 🔄 Konversi antar satuan suhu.
* 📊 Menampilkan hasil konversi.
* 💻 Berjalan melalui terminal / command prompt.

# ▶️ Cara Menjalankan

### 1. Clone Repository

```bash
git clone https://github.com/Diann13/Konversi_suhu_menggunakan_dart.git
```

---

### 2. Install Dependency

```bash
dart pub get
```

---

### 3. Jalankan Program

```bash
dart run
```

atau

```bash
dart nama_file.dart
```

# 🌡️ Contoh Konversi

Sebagai contoh:

```text
Input

Suhu : 100°C
```

Output yang diharapkan dapat berupa:

```text
Fahrenheit : 212°F

Kelvin : 373.15 K

Reamur : 80°R
```

*(Contoh di atas hanya ilustrasi hasil konversi.)*
